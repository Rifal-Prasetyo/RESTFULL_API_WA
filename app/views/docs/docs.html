Sending Message

Kirim Pesan

Endpoint: /whatsapp/sendmessage

Method: POST

Request 


            {
                "api_key": "APALOBUKABUKA|BIKINADMINMARAH", // your Screet key
                "receiver": "08200000000", // target
                "type": "PERSONAL", // PERSONAL | GROUP
                "data": {
                  "message": "Hai kamu yang manis" // message
                }
            }

Respone 


          {
              "code": 200 // 200 | 500 | 404,
              "status": true // boolean,
              "message": "Alhamdulillah" // message from server
          }

Sending Media

Kirim Media berupa gambar video

Endpoint: /whatsapp/sendimage

Method: POST

Request 


            {
                "api_key": "APALOBUKABUKA|BIKINADMINMARAH", // your Screet key
                "receiver": "08200000000", // target
                "type": "PERSONAL", // PERSONAL | GROUP
                "media": "image", // image | video 
                "data": {
                  "url": "http://example.com/image.jpg", // url image 
                  "caption": "Gambar ini" // caption
                }
            }

Untuk Mendapatkan ID Grub ikuti panduan ini Klik Disini

Respone 


          {
              "code": 200 // 200 | 500 | 404,
              "status": true // boolean,
              "message": "Alhamdulillah" // message from server
          }

Whatsapp Command

Perintah Whatsapp

Perintah Whatsapp dapat dilakukan dengan melakukan Reply ke Bot Whatsapp dengan memasukkan perintah tertentu, biasanya diawali dengan tanda /[command] [Argument]. berikut adalah perintah yang dapat dilakukan ke bot Whatsapp

Cek Status

Cek Status dasar Identitas mu dan penggunaan kamu

Mendapatkan ID Group

Mendapatkan Id Grub untuk melakukan automisasi Bot ke Grub kamu

Langkah: Undang Bot Whatsapp kedalam Grub yang dituju, masukkan perintah /capt di dalam group

Setelah memasukkan perintah tersebut otomatis bot mengirimkan chat ke kamu secara pribadi
